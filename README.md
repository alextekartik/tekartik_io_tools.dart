# tekartik_io_tools

Common io tools

    pub global activate -s git git://github.com/alextekartik/tekartik_io_tools.dart

# Commands

## rscpull

    rscpull

Recursively call 'git pull' and 'hg pull -u'

## rscstatus

    rscstatus

Recursively get git or hg status for all source control project found
